# 基础知识
1. 常数项级数的概念与性质
    1. 概念
    ```tex
    \displaystyle
    \footnotesize
    形式: \sum_{n = 1}^{\infty} u_n = u_1 + u_2 + ... + u_n + ..., 叫做无穷级数, 简称级数  \\
    u_i是通项, 且为常数, 则\sum_{n = 1}^{\infty} u_n就可以叫做常数项(无穷)级数  \\
    S_n = u_1 + u_2 + ... + u_n, S_n就是级数的部分和, \{S_n\}是级数的部分和数列  \\
    由于\lim\limits_{n \to \infty} S_n = \sum_{n = 1}^{\infty} u_n, 故可以用极限处理无穷级数  \\
    \lim\limits_{n \to \infty} S_n = S, 则\sum_{n = 1}^{\infty} u_n收敛, 极限值即为收敛级数\sum_{n = 1}^{\infty} u_n的和  \\
    \lim\limits_{n \to \infty} S_n不存在或为\pm\infty, 则\sum_{n = 1}^{\infty} u_n发散  \\
    研究\sum_{n = 1}^{\infty} u_n的收敛或发散, 也可以叫做研究\sum_{n = 1}^{\infty} u_n的敛散性  \\
    去掉级数前m项, \sum_{n = m+1}^{\infty}u_n = u_{m + 1} + u_{m + 2} + ...叫做级数\sum_{n = 1}^{\infty}u_n的m项后余项  \\
    ```
    2. 性质
    ```tex
    \displaystyle
    \footnotesize
    部分和取极限存在,级数收敛; 级数收敛, 部分和取极限存在  \\
    (1) 若级数\sum_{n = 1}^{\infty} u_n收敛, 且和为S, 则给任意常数a, \sum_{n = 1}^{\infty} au_n也收敛,和为aS, 即  \\
    \sum_{n = 1}^{\infty} au_n = a\sum_{n = 1}^{\infty} u_n  \\
    证明: \sum_{n = 1}^{\infty} au_n的部分和为au_1 + ... + au_n = a(u_1 + ... + u_n) = aS_n  \\
    \lim\limits_{n \to \infty} S_n = \sum_{n = 1}^{\infty} u_n = S, 故\lim\limits_{n \to \infty}aS_n = aS  \\
    故\sum_{n = 1}^{\infty} au_n = aS = a\sum_{n = 1}^{\infty} u_n  \\
    (2) 若级数\sum_{n = 1}^{\infty} u_n, \sum_{n = 1}^{\infty} v_n均收敛, 其和分别为S, T, 则
    \sum_{n = 1}^{\infty} (u_n \pm v_n)收敛,和为S \pm T, 即  \\
    \sum_{n = 1}^{\infty} (u_n \pm v_n) = \sum_{n = 1}^{\infty} u_n \pm \sum_{n = 1}^{\infty} v_n  \\
    证明: \sum_{n = 1}^{\infty} (u_n \pm v_n)的部分和为(u_1 \pm v_1)  ... + (u_n \pm v_n) = 
    (u_1 + ... + u_n) \pm (v_1 + ... + v_n) = S_n \pm T_n  \\
    = \lim\limits_{n \to \infty} (S_n \pm T_n) = \lim\limits_{n \to \infty} S_n \pm \lim\limits_{n \to \infty} T_n = S \pm T  \\
    (3) 若级数\sum_{n = 1}^{\infty} u_n, \sum_{n = 1}^{\infty} v_n均收敛, 其和分别为S, T, 则任给常数a, b, 有\sum_{n = 1}^{\infty} (au_n + bv_n)收敛, 和为aS + bT, 即  \\
    \sum_{n = 1}^{\infty} (au_n + bv_n) = a\sum_{n = 1}^{\infty} u_n + b\sum_{n = 1}^{\infty} v_n  \\
    (4) 若级数\sum_{n = 1}^{\infty} u_n收敛, 其任意m项后余项\sum_{n = m + 1}^{\infty} u_n收敛; 若级数m项后余项\sum_{n = m + 1}^{\infty} u_n收敛, 级数\sum_{n = 1}^{\infty} u_n收敛  \\
    证明: \sum_{n = m + 1}^{\infty} u_n的部分和为T_k = u_{m+1} + ... + u_{m+k}  \\
    T_k = S_{m + k} - S_{m}, 因为\sum_{n = 1}^{\infty} u_n收敛, \lim\limits_{k \to \infty}S_{m + k} = S  \\
    故\lim\limits_{k \to \infty}T_k = \lim\limits_{k \to \infty}S_{m + k} - S_{m} = S - S_{m}(存在), 故级数收敛  \\
    而\sum_{n = m + 1}^{\infty} u_n收敛, \lim\limits_{k \to \infty}T_k = T(存在), \lim\limits_{k \to \infty}T_k = \lim\limits_{k \to \infty}S_{m + k} - S_{m}  \\
    \lim\limits_{n \to \infty}S_n = \lim\limits_{k \to \infty}S_{m + k} = T + S_m(存在), 级数\sum_{n = 1}^{\infty} u_n收敛  \\
    添加或删除级数前面任意有限项, 不会改变级数的敛散性  \\
    (5) 若级数\sum_{n = 1}^{\infty} u_n收敛, 则\lim\limits_{n \to \infty} u_n = 0  \\
    证明: u_n = S_n - S_{n - 1}, 故\lim\limits_{n \to \infty} u_n = \lim\limits_{n \to \infty} S_n - \lim\limits_{n \to \infty}S_{n - 1} = S - S = 0  \\
    这里只有单向, 级数收敛, 尾项极限趋向0. 若尾项极限趋向0, 级数不一定收敛  \\
    逆否命题: 若\lim\limits_{n \to \infty} u_n \mathrlap{\,/}{=} 0, 则\sum_{n = 1}^{\infty} u_n必发散  \\
    ```
2. 级数敛散性的判别方法
    1. 正项级数及敛散性判别
    ```tex
    \displaystyle
    \footnotesize
    通项u_n \geqslant 0, 则\sum_{n = 1}^{\infty} u_n为正项级数  \\
    (1) 收敛原则  \\
    正项级数\sum_{n = 1}^{\infty} u_n收敛的充要条件是他的部分和数列\{S_n\}有界  \\
    正项级数\sum_{n = 1}^{\infty} u_n发散的充要条件是他的部分和数列\{S_n\}无界  \\
    证明: 必要性  \\
    u_n \geqslant 0, 故S_n = u_1 + ... + u_n \geqslant 0, 且S_1 \leqslant S_2 \leqslant ... \leqslant S_n \leqslant ...  \\
    当\sum_{n = 1}^{\infty} u_n收敛, \lim\limits_{n \to \infty}S_n存在, \{S_n\}有上界. \{S_n\}上下界都存在, 故有界  \\
    充分性  \\
    \{S_n\}有界, 且\{S_n\}单调不减, 根据单调有界准则, \{S_n\}收敛, 即\lim\limits_{n \to \infty}S_n存在, \sum_{n = 1}^{\infty} u_n收敛  \\
    正项级数中, 数列\{S_n\}单调不减, 若\{S_n\}有界, \lim\limits_{n \to \infty} S_n = S, 若\{S_n\}无界, \lim\limits_{n \to \infty} S_n = +\infty  \\
    (2) 比较判别法  \\
    两个正项级数\sum_{n = 1}^{\infty} u_n, \sum_{n = 1}^{\infty} v_n, 从某项起, u_n \leqslant v_n, 则  \\
    \sum_{n = 1}^{\infty} u_n 发散, \sum_{n = 1}^{\infty} v_n 发散  \\
    \sum_{n = 1}^{\infty} v_n 收敛, \sum_{n = 1}^{\infty} u_n 收敛  \\
    小发散, 大发散; 大收敛, 小收敛  \\
    证明: 因为去掉级数的前有限项不改变级数的敛散性, 则可设对一切n有u_n \leqslant v_n,  \\
    记\sum_{n = 1}^{\infty} u_n, \sum_{n = 1}^{\infty} v_n的部分和为A_n = \sum_{k = 1}^{n} u_k, B_n = \sum_{k = 1}^{n} v_k, 则A_n \leqslant B_n  \\
    若\sum_{n = 1}^{\infty} v_n收敛, 则\{B_n\}有界, A_n \leqslant B_n \leqslant M, 故\{A_n\}有界, \sum_{n = 1}^{\infty} u_n 收敛  \\
    若\sum_{n = 1}^{\infty} u_n 发散, 则\{A_n\}无界, \lim\limits_{n \to \infty} A_n = +\infty, 又A_n \leqslant B_n, 故
    \lim\limits_{n \to \infty} B_n = +\infty, 则\{B_n\}无界, \sum_{n = 1}^{\infty} v_n发散  \\
    (3) 比较判别法的极限形式  \\
    两个正项级数\sum_{n = 1}^{\infty} u_n, \sum_{n = 1}^{\infty} v_n, v_n \mathrlap{\,/}{=} 0, 且\lim\limits_{n \to \infty} \frac{u_n}{v_n} = A  \\
    (一)当A = 0, \sum_{n = 1}^{\infty} v_n收敛, \sum_{n = 1}^{\infty} u_n收敛  \\
    (二)当A = +\infty, \sum_{n = 1}^{\infty} v_n发散, \sum_{n = 1}^{\infty} u_n发散  \\
    (三)当0 < A < +\infty, \sum_{n = 1}^{\infty} u_n, \sum_{n = 1}^{\infty} v_n有相同敛散性  \\
    证明: 因为u_n \geqslant 0, v_n > 0, \lim\limits_{n \to \infty} \frac{u_n}{v_n} = A \geqslant 0  \\
    A = 0, 根据保号性 n \to \infty, \frac{u_n}{v_n} < 1, u_n < v_n, 大收敛小收敛  \\
    A = +\infty, 根据保号性 n \to \infty, \frac{u_n}{v_n} > 1, u_n > v_n, 小发散大发散  \\
    第三个根据极限定义有, \forall \epsilon > 0 , \exists N > 0, 当n > N时, |\frac{u_n}{v_n} - A| < \epsilon  \\
    取\epsilon = \frac{A}{2}, \frac{A}{2}v_n < u_n < \frac{3A}{2}v_n   \\
    若v_n发散, \frac{A}{2}v_n 发散, u_n发散, 若v_n收敛, \frac{3A}{2}v_n 收敛, u_n 收敛  \\
    (4) 比值判别法(达朗贝尔判别法)  \\
    有正项级数\sum_{n = 1}^{\infty} u_n, 如果\lim\limits_{n \to \infty} \frac{u_{n + 1}}{u_n} = p, 则  \\
    若p < 1, \sum_{n = 1}^{\infty} u_n收敛  \\
    若p > 1, \sum_{n = 1}^{\infty} u_n发散  \\
    若p = 1, 无法判断级数敛散性  \\
    证明  \\
    (一)当p < 1, 取\epsilon >0且充分小, 使 p + \epsilon < 1, 故\exists N > 0, 当n > N, 有|\frac{u_{n + 1}}{u_n} - p| < \epsilon  \\
    \frac{u_{n + 1}}{u_n} < \epsilon + p < 1  \\
    u_{n + 1} < u_n(\epsilon + p) < ... <  u_1(\epsilon + p)^n = v_n  \\
    算v_n的部分和的极限, 极限存在, 故\sum_{n = 1}^{\infty} v_n收敛, 大收敛小收敛, 故\sum_{n = 1}^{\infty} u_{n+1}收敛, 即\sum_{n = 1}^{\infty} u_n收敛  \\
    (二)当p > 1, 取\epsilon >0且充分小, 使 p - \epsilon > 1, 故\exists N > 0, 当n > N, 有|\frac{u_{n + 1}}{u_n} - p| < \epsilon  \\
    \frac{u_{n + 1}}{u_n} > p - \epsilon > 1  \\
    u_{n + 1} > (p - \epsilon)u_n > ... > (p - \epsilon)^n u_1 = v_n  \\
    算v_n的部分和极限, 极限为+\infty, 故\sum_{n = 1}^{\infty} v_n发散, 小发散大发散, 故\sum_{n = 1}^{\infty} u_{n+1}发散, 即\sum_{n = 1}^{\infty} u_n发散(u_{n+1}和u_n是没有区别的)  \\
    (三)对于\sum_{n = 1}^{\infty} \frac{1}{n}, \sum_{n = 1}^{\infty} \frac{1}{n^2}, 前者发散,后者收敛,但
    \lim\limits_{n \to \infty} \frac{u_{n+1}}{u_n}都为1  \\
    \sum_{n = 1}^{\infty} \frac{1}{n} = 1 + (\frac12 + \frac13 + ... + \frac{1}{10}) + (\frac{1}{11} + \frac{1}{12} + ... + \frac{1}{100}) + ... > 1 + \frac{9}{10} + \frac{90}{100} + \frac{900}{1000} + ... = 1 + \frac{9}{10} + \frac{9}{10} ...  =
    1 + \infty = \infty  \\
    故\sum_{n = 1}^{\infty} \frac{1}{n}的部分和极限为+\infty, \sum_{n = 1}^{\infty} \frac{1}{n}发散  \\
    \sum_{n = 1}^{\infty} \frac{1}{n^2} = 1 + \sum_{n = 2}^{\infty} \frac{1}{n(n - 1)} = 1 + \sum_{n = 2}^{\infty} (\frac{1}{n - 1} - \frac{1}{n})  \\ 
    \sum_{n = 2}^{\infty} (\frac{1}{n - 1} - \frac{1}{n})的部分和极限为1, 故\sum_{n = 2}^{\infty} (\frac{1}{n - 1} - \frac{1}{n})收敛, 1 + \sum_{n = 2}^{\infty} (\frac{1}{n - 1} - \frac{1}{n})收敛, \sum_{n = 1}^{\infty} \frac{1}{n^2}收敛  \\
    (5) 根值判别法(柯西判别法)  \\
    有正项级数\sum_{n = 1}^{\infty} u_n, 如果\lim\limits_{n \to \infty} \sqrt[n]{u_n} = p, 则  \\
    (一) 若p < 1, \sum_{n = 1}^{\infty} u_n收敛  \\
    (二) 若p > 1, \sum_{n = 1}^{\infty} u_n发散  \\
    证明  \\
    (一)当p < 1, 取\epsilon >0且足够小, 使 p + \epsilon < 1, 则 \exists N > 0, 当n > N时, 有 |\sqrt[n]{u_n} - p| < \epsilon  \\
    u_n < (p + \epsilon)^n = v_n  \\
    \sum_{n = 1}^{\infty} v_n收敛, 故\sum_{n = 1}^{\infty} u_n收敛  \\
    (二)当p > 1, 取\epsilon >0且足够小, 使 p - \epsilon > 1, 则 \exists N > 0, 当n > N时, 有 |\sqrt[n]{u_n} - p| < \epsilon  \\
    u_n > (p - \epsilon)^n = v_n  \\
    \sum_{n = 1}^{\infty} v_n发散, 故\sum_{n = 1}^{\infty} u_n发散  \\
    (三)对于\sum_{n = 1}^{\infty} \frac{1}{n}发散, \sum_{n = 1}^{\infty} \frac{1}{n^2}收敛, 有
    \lim\limits_{n \to \infty} \sqrt[n]{\frac{1}{n}} = 1, \lim\limits_{n \to \infty} \sqrt[n]{\frac{1}{n^2}} = 1  \\
    这里因为n最小为1, -nlnn最多到0, 所以极限才到了1  \\
    ```
    2. 交错级数及敛散性判别
    ```tex
    \displaystyle
    \footnotesize
    交错级数: 级数各项正负相间出现  \\
    一般为\sum_{n = 1}^{\infty}(-1)^{n - 1}u_n = u_1 - u_2 + u_3 + (-1)^{n - 1}u_n + ..., (u_n > 0)  \\
    莱布尼茨判别法: 有交错级数\sum_{n = 1}^{\infty}(-1)^{n - 1}u_n, \{u_n\}单调不增且\lim\limits_{n \to \infty} u_n = 0, 则级数收敛  \\
    证明  \\
    级数前2n项部分和为S_{2n} = (u_1 - u_2) + (u_3 - u_4) + ... + (u_{2n - 1} - u_{2n})  \\
    因为\{u_n\}单调不增, u_{2n - 1} - u_{2n} \geqslant 0, S_{2n} \geqslant S_{2n - 2}, \{S_{2n}\}单调不减  \\
    S_{2n} = u_1 - (u_2 - u_3) - ... - (u_{2n - 2} - u_{2n - 1}) - u_{2n}, s_{2n} \leqslant u_1, \{S_{2n}\}有上界u_1  \\ 
    根据单调有界准则, \{S_{2n}\}收敛, \lim\limits_{n \to \infty}S_{2n}存在  \\
    S_{2n+1} = S_{2n} + u_{2n+1}, 因为\lim\limits_{n \to \infty}u_n = 0, \lim\limits_{n \to \infty}S_{2n+1}存在  \\
    根据\orange{一个数列的奇数列偶数列分别收敛到同一个数, 则该数列收敛}, 故\lim\limits_{n \to \infty}S_n存在, 故\sum_{n = 1}^{\infty}(-1)^{n - 1}u_n收敛  \\
    ```
    3. 任意项级数及其敛散性判别
    ```tex
    \displaystyle
    \footnotesize
    任意项级数: 级数各项可正, 可负, 可0  \\
    形如\sum_{n = 1}^{\infty} u_n  \\
    处理的时候写成\sum_{n = 1}^{\infty} |u_n|, 从而得到了正项级数, 叫做原级数\sum_{n = 1}^{\infty} u_n的绝对值级数  \\
    正项级数的敛散性判别法可用于绝对值级数  \\
    以下为绝对值级数\sum_{n = 1}^{\infty} |u_n|与原级数\sum_{n = 1}^{\infty} u_n的关系定义  \\
    (1) 设\sum_{n = 1}^{\infty} u_n为任意项级数, 若\sum_{n = 1}^{\infty} |u_n|收敛, \sum_{n = 1}^{\infty} u_n绝对收敛  \\
    (2) 设\sum_{n = 1}^{\infty} u_n为任意项级数, \sum_{n = 1}^{\infty} u_n收敛, \sum_{n = 1}^{\infty} |u_n|发散, 则\sum_{n = 1}^{\infty} u_n条件收敛  \\
    绝对值级数与任意项级数的定理  \\
    (1) 若\sum_{n = 1}^{\infty} |u_n|收敛(即\sum_{n = 1}^{\infty} u_n绝对收敛), \sum_{n = 1}^{\infty} u_n必收敛  \\
    证明: \sum_{n = 1}^{\infty} u_n = u_1 + u_2 + ... + u_n + ...  \\
    \sum_{n = 1}^{\infty} u_n的正项和为\sum_{k = 1}^{\infty} a_k = a_1 + a_2 + ... + a_k + ...  \\
    \sum_{n = 1}^{\infty} u_n的负项加绝对值和为\sum_{i = 1}^{\infty} b_i = b_1 + b_2 + ... + b_i + ...  \\
    故\sum_{n = 1}^{\infty} u_n = \sum_{k = 1}^{\infty} a_k - \sum_{i = 1}^{\infty} b_i, \sum_{n = 1}^{\infty} u_n部分和为  \\
    S_n = A_k - B_i  \\
    \sum_{n = 1}^{\infty} |u_n| = \sum_{k = 1}^{\infty} a_k + \sum_{i = 1}^{\infty} b_i, 且\sum_{n = 1}^{\infty} |u_n|收敛  \\
    \sum_{k = 1}^{\infty} a_k, \sum_{i = 1}^{\infty} b_i也都收敛, 故\lim\limits_{n \to \infty}S_n = \lim\limits_{n \to \infty}A_k - \lim\limits_{n \to \infty}B_i, 极限存在  \\
    故级数\sum_{n = 1}^{\infty} u_n收敛  \\
    ```
    4. 收敛级数的性质
    ```tex
    \displaystyle
    \footnotesize
    (1) 收敛级数的项任意加括号后所得新级数仍收敛, 和不变  \\
    证明: 设级数\sum_{n = 1}^{\infty} u_n收敛, 部分和\{S_n\}的极限存在, 级数任意项加括号后为  \\
    (u_1 + ... + u_{n_1}) + (u_{n_1 + 1} + ... + u_{n_2}) + ... + (u_{n_k + 1} + ... + u_{n_{k + 1}}) + ... \\
    其部分和数列为\{S_{n_k}\}, k = 1, 2 ...  \\
    \{S_{n_k}\}是\{S_{n}\}的子列, 故\{S_{n_k}\}也收敛, 其极限存在,故加括号后的级数仍收敛  \\
    (原级数收敛, 部分和数列有界, 加括号后的级数部分和数列为原部分和数列子集, 走向和收敛值相同)  \\
    该命题逆否命题: 若加括号后的新级数发散, 原级数必然发散  \\
    若加括号后的新级数收敛,原级数不一定收敛, 交错级数\sum_{n = 1}^{\infty} (-1)^{n - 1}a (a不为0), 两两相加收敛, 原级数发散  \\ 
    但若加括号后的新级数收敛, 且原级数通项极限为0, 原级数收敛  \\
    (2) 若原级数绝对收敛, 不论将各项如何重新排列, 所得新级数也绝对收敛, 和不变(绝对收敛的级数有可交换性)  \\
    先设正项级数\sum_{n = 1}^{\infty} u_n, u_n \geqslant 0, 和收敛于S  \\
    将其任意项重排得到\sum_{n = 1}^{\infty} \widetilde{u_n}, 部分和T_n \leqslant S, 由收敛原则(正项级数部分和数列单调不减), 
    \sum_{n = 1}^{\infty} \widetilde{u_n} = \widetilde{S} \leqslant S   \\ 
    \sum_{n = 1}^{\infty} u_n又可看做\sum_{n = 1}^{\infty} \widetilde{u_n}的重排级数, 故S \leqslant \widetilde{S}  \\
    这里表明收敛的正项级数任意交换各项顺序得到的新级数仍收敛, 和不变  \\
    再设任意项级数\sum_{n = 1}^{\infty}u_n绝对收敛, \sum_{n = 1}^{\infty}|u_n|收敛, \sum_{n = 1}^{\infty} |\widetilde{u_n}|收敛, \sum_{n = 1}^{\infty} \widetilde{u_n}绝对收敛  \\
    \sum_{n = 1}^{\infty}u_n重排后的\sum_{n = 1}^{\infty} \widetilde{u_n}仍是原来的数,故和不变  \\
    ```

3. 阿贝尔定理与幂级数的收敛域
    1. 相关概念
    ```tex
    \displaystyle
    \footnotesize
    (1) 函数项级数  \\
    函数项级数: \sum_{n = 1}^{\infty}u_n(x)  \\
    当x取x_0, \sum_{n = 1}^{\infty}u_n(x_0)是常数项级数  \\
    (2) 幂级数  \\
    幂级数: \sum_{n = 1}^{\infty}u_n(x)的一般项是n次幂函数  \\
    一般形式为\sum_{n = 0}^{\infty}a_n(x - x_0)^n \to (a_n是幂级数的系数)  \\
    标准形式为\sum_{n = 0}^{\infty}a_nx^n \to (a_n是幂级数的系数)  \\
    (3) 收敛点与发散点  \\
    \sum_{n = 1}^{\infty}u_n(x)的收敛点: x_0 \in I , \sum_{n = 1}^{\infty}u_n(x_0)收敛  \\
    \sum_{n = 1}^{\infty}u_n(x)的发散点: x_0 \in I , \sum_{n = 1}^{\infty}u_n(x_0)发散  \\
    (4) 收敛域  \\
    收敛域: 函数项级数\sum_{n = 1}^{\infty}u_n(x)所有收敛点的集合  \\
    ```
    2. 阿贝尔定理  
    ```tex
    \displaystyle
    \footnotesize
    (1) 阿贝尔定理内容  \\
    当幂级数\sum_{n = 0}^{\infty}a_nx^n在点x = x_1(x_1不为0)收敛, 对于|x| < |x_1|的一切x, 幂级数绝对收敛  \\
    当幂级数\sum_{n = 0}^{\infty}a_nx^n在点x = x_2(x_2不为0)发散, 对于|x| > |x_2|的一切x, 幂级数发散  \\
    (2) 收敛半径存在性  \\
    幂级数\sum_{n = 0}^{\infty}a_nx^n的收敛半径R(R \geqslant 0)必存在, 且  \\
    (一)仅在点x = 0收敛, 收敛半径R = 0  \\
    (二)在整个数轴都收敛, 收敛半径R = +\infty  \\
    (三)当|x| < R, 幂级数绝对收敛, 且当|x| > R, 幂级数发散, 收敛半径为R  \\
    当x = R与x = -R时, 幂级数可能收敛也可能发散  \\
    (3) 收敛半径的求法  \\
    若\lim\limits_{n \to \infty} |\frac{a_{n+1}}{a_n}| = p, 则\sum_{n = 0}^{\infty}a_nx^n收敛半径R为
    R = 
    \begin{cases}
    \frac{1}{p}, p不为0  \\
    \,+\infty, p = 0  \\
    0, p = +\infty  \\
    \end{cases}  \\
    若\sum_{n = 0}^{\infty}a_nx^{kn + l}, 半径R = \sqrt[k]{\frac{1}{p}}  \\
    (4) 收敛区间与收敛域  \\
    开区间(-R, R)为幂级数的\sum_{n = 0}^{\infty}a_nx^n的收敛区间, 找到x = \pm R处的敛散性就可以确定其收敛域  \\
    因为收敛域可能包含x = \pm R的一个或两个点,也可能不包括  \\
    注: 标准幂函数和一般幂级数都可用上述定理, 一般幂级数\sum_{n = 0}^{\infty}a_n(x - x_0)^n在x = x_1收敛, 则|x - x_0| < |x_1 - x_0|处也收敛  \\
    ```

    3. 幂级数的四则运算
    ```tex
    \displaystyle
    \footnotesize
    若幂级数\sum_{n = 0}^{\infty}a_nx^n, \sum_{n = 0}^{\infty}b_nx^n的收敛半径分别为R_a, R_b(R_a\mathrlap{\,/}{=}R_b)则  \\
    (1) k\sum_{n = 0}^{\infty}a_nx^n = \sum_{n = 0}^{\infty}ka_nx^n, |x| < R_a  \\
    (2) \sum_{n = 0}^{\infty}a_nx^n \pm \sum_{n = 0}^{\infty}b_nx^n = \sum_{n = 0}^{\infty}(a_n \pm b_n)x^n, |x| < R  \\
    (3) (\sum_{n = 0}^{\infty}a_nx^n)(\sum_{n = 0}^{\infty}b_nx^n) = \sum_{n = 0}^{\infty}c_nx^n  \\
    R = min\{R_a, R_b\}, c_n = \sum_{k = 0}^na_kb_{n - k}  \\
    幂级数下标和通项的几种变化(化简成一项一项的还是原来的数据)  \\
    (一)\sum_{n = k}^{\infty}a_nx^n = \sum_{n = k + l}^{\infty}a_{n - l}x^{n - l}  \\
    (二)\sum_{n = k}^{\infty}a_nx^n = a_kx^k + ... + a_{k + l - 1}x^{k + l - 1} + \sum_{n = k + l}^{\infty}a_nx^n  \\
    (三)\sum_{n = k}^{\infty}a_nx^n = x^l\sum_{n = k}^{\infty}a_nx^{n - l}  \\
    如\sum_{n = 0}^{\infty}a_nx^{2n} + \sum_{n = 0}^{\infty}b_{n+1}x^{2n+2} = a_0 + \sum_{n = 1}^{\infty}a_nx^{2n} +
    \sum_{n = 1}^{\infty}b_nx^{2n} = a_0 + \sum_{n = 1}^{\infty}(a_n + b_n)x^{2n}  \\
    ```
    4. 幂级数的性质
    ```tex
    \displaystyle
    \footnotesize
    (1) 幂级数\sum_{n = 0}^{\infty}a_nx^{n}的和函数S(x)在收敛区间I连续, 如果幂级数在收敛区间端点x = R(或x = -R)收敛,  \\
    则和函数S(x)在(-R, R](或[-R, R))连续  \\
    (2) 幂级数\sum_{n = 0}^{\infty}a_nx^{n}的和函数S(x)在收敛域I可积, 且有逐项积分公式  \\
    \int_0^xS(t)dt = \int_0^x(\sum_{n = 0}^{\infty}a_nt^{n})dt = \sum_{n = 0}^{\infty}a_n\int_0^xt^ndt =
    \sum_{n = 0}^{\infty}\frac{a_n}{n+1}x^{n+1} (x \in I)  \\
    逐项积分后的幂级数收敛半径与原级数相同, 收敛域可能扩大  \\
    (3) 幂级数\sum_{n = 0}^{\infty}a_nx^{n}的和函数S(x)在收敛区间(-R, R)可导, 且逐项求导公式为  \\
    S'(x) = (\sum_{n = 0}^{\infty}a_nx^{n})' = \sum_{n = 0}^{\infty}(a_nx^{n})' = \sum_{n = 0}^{\infty}na_nx^{n - 1}(|x| < R)  \\
    逐项求导后的幂级数与原级数有相同收敛半径,但收敛域可能减小  \\
    级数的积分号(求导号)和求和号可以换位置  \\
    ```

6. 傅里叶级数
```tex
\displaystyle
\footnotesize
(1)基本知识  \\
三角函数: y = Asin(\omega t + \varphi), \omega是角频率, A是振幅, \varphi是初相角, \omega = \frac{2\pi}{T}  \\
三角函数族:\{1, cosx, sinx, cos2x, sin2x, ..., cosnx, sinnx, ...\}  \\
三角函数族的正交性: 三角函数族中任意两个不同函数的乘积在[-\pi, \pi]的积分值为0  \\
\int_{-\pi}^\pi 1 * cosnxdx = \frac{sinnx}{n}|_{-\pi}^{\pi} = 0  \\
\int_{-\pi}^\pi 1 * sinnxdx = -\frac{cosnx}{n}|_{-\pi}^{\pi} = 0  \\
\int_{-\pi}^\pi cosnx * sinmxdx = \frac12\int_{-\pi}^\pi [sin(m + n)x + sin(m - n)x]dx = 0 (m和n可以相等) \\
\int_{-\pi}^\pi cosnx * cosmxdx = \frac12\int_{-\pi}^\pi [cos(m + n)x + cos(m - n)x]dx = 0 (m\mathrlap{\,/}{=} n)  \\
\int_{-\pi}^\pi sinnx * sinmxdx = \frac12\int_{-\pi}^\pi [cos(m - n)x - cos(m + n)x]dx = 0 (m\mathrlap{\,/}{=} n)  \\
当m = n时  \\
\int_{-\pi}^\pi cosnx * cosmxdx = \int_{-\pi}^\pi \frac{1 + cos2nx}{2} dx = \pi  \\
\int_{-\pi}^\pi sinnx * sinmxdx = \int_{-\pi}^\pi \frac{1 - cos2nx}{2} dx = \pi  \\
(2)周期函数可由序列的正弦型函数叠加得到  \\
\varphi(t) = A_0 + \sum_{n = 1}^\infty A_nsin(n\omega t + \varphi_n)叫做三角级数  \\
(3)三角级数的变化得到傅里叶级数  \\
\varphi(t) = A_0 + \sum_{n = 1}^\infty A_nsin(n\omega t + \varphi_n)
= A_0 + \sum_{n = 1}^\infty(A_nsin(n\omega t)cos\varphi_n + A_ncos(n\omega t)sin\varphi_n)  \\
令a_n = A_nsin\varphi_n, b_n = A_ncos\varphi_n, x = \omega t = \frac{2\pi t}{T}  \\
f(x) = A_0 + \sum_{n = 1}^\infty (a_ncos nx + b_nsin nx), 周期为2\pi  \\
对f(x)在[-\pi, \pi]积分得到, \int_{-\pi}^{\pi} f(x)dx = 2\pi A_0  \\
A_0 = \frac{1}{2\pi}\int_{-\pi}^{\pi} f(x)dx  \\
对f(x)乘cosmx再在[-\pi, \pi]积分得到, \int_{-\pi}^{\pi} f(x)cosmxdx = a_n\pi(仅在m = n时, cosnxcosmx的积分不为0)  \\
a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)cosmxdx = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)cosnxdx  \\
对f(x)乘sinmx再在[-\pi, \pi]积分得到, \int_{-\pi}^{\pi} f(x)sinmxdx = b_n\pi(仅在m = n时, cosnxcosmx的积分不为0)  \\
b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)sinmxdx = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)sinnxdx  \\
当n = 0时, a_0 = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)dx, 故A_0 = \frac{a_0}{2}  \\
\red{得到[-\pi, \pi]的傅里叶级数}  \\
f(x) = \frac{a_0}{2} + \sum_{n = 1}^\infty (a_ncos nx + b_nsin nx)  \\
a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)cosnxdx, n = 0, 1, 2...  \\
b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} f(x)sinnxdx, n = 1, 2, 3...  \\
(4)傅里叶级数的收敛性\to狄利克雷收敛定理(周期函数满足收敛性则可用傅里叶级数)  \\
f(x)在[-\pi, \pi]连续或只有有限个第一类间断点, 且最多有有限个(真正的)极值点, 则f(x)的傅里叶级数在[-\pi, \pi]处处收敛  \\
和函数S(x) = \frac{a_0}{2} + \sum_{n = 1}^\infty (a_ncos nx + b_nsin nx)  \\
且S(x) = \begin{cases}
f(x), x为连续点  \\
\frac12(\lim\limits_{x \to x_0^+}f(x) + \lim\limits_{x \to x_0^-}f(x)) ,x为间断点  \\
\frac12(\lim\limits_{x \to -\pi^+}f(x) + \lim\limits_{x \to \pi^-}f(x)) ,x = \pm \pi  \\
\end{cases}  \\
只有x为连续点f(x) = S(x), 间断点和端点可能不相等, 故只能是f(x) \enspace\text{\textasciitilde}\enspace S(x)  \\
因为傅里叶级数以2\pi为周期, 故可以将展开拓展到整个数轴(改一下x的取值范围即可)  \\
(5)任意区间的傅里叶展开式  \\
区间端点的变化: \int_{a}^{a + 2\pi} f(x)dx =\int_{-\pi}^{\pi} f(x)dx  \\
区间长度的变化: 若-l \leqslant x \leqslant l, 则-\pi \leqslant \frac{\pi x}{l} = t \leqslant \pi, 故  \\
f(x) = f(\frac{lt}{\pi}) = g(t)  \\
g(t) \enspace\text{\textasciitilde}\enspace \frac{a_0}{2} + \sum_{n = 1}^\infty (a_ncos nt + b_nsin nt)  \\
a_n = \frac{1}{\pi}\int_{-\pi}^{\pi} g(t)cosntdt  \\
b_n = \frac{1}{\pi}\int_{-\pi}^{\pi} g(t)sinntdt  \\
故f(x) \enspace\text{\textasciitilde}\enspace \frac{a_0}{2} + \sum_{n = 1}^\infty (a_ncos \frac{n\pi x}{l} + b_nsin \frac{n\pi x}{l})  \\
a_n = \frac{1}{l}\int_{-l}^{l} f(x)cos\frac{n\pi x}{l}dx  \\
b_n = \frac{1}{l}\int_{-l}^{l} f(x)sin\frac{n\pi x}{l}dx  \\
若f(x)在[-l, l]连续或有有限个第一类间断点, 有有限个真正极大值, 则f(x)的傅里叶级数在[-l, l]处处收敛,  \\
和函数S(x) = \frac{a_0}{2} + \sum_{n = 1}^\infty (a_ncos \frac{n\pi x}{l} + b_nsin \frac{n\pi x}{l})且  \\
S(x) = \begin{cases}
f(x), x为连续点  \\
\frac12(\lim\limits_{x \to x_0^+}f(x) + \lim\limits_{x \to x_0^-}f(x)) ,x为间断点  \\
\frac12(\lim\limits_{x \to -l^+}f(x) + \lim\limits_{x \to l^-}f(x)) ,x = \pm l  \\
\end{cases}  \\
(6)正弦级数与余弦级数  \\
在[-l, l], f(x)为奇函数, 则a_n = \frac{1}{l}\int_{-l}^{l} f(x)cos\frac{n\pi x}{l}dx = 0  \\
f(x) \enspace\text{\textasciitilde}\enspace \sum_{n = 1}^\infty b_nsin \frac{n\pi x}{l}, 展开式只有正弦函数, 故为正弦级数  \\
在[-l, l], f(x)为偶函数, 则b_n = \frac{1}{l}\int_{-l}^{l} f(x)sin\frac{n\pi x}{l}dx = 0  \\
f(x) \enspace\text{\textasciitilde}\enspace \frac{a_0}{2} + \sum_{n = 1}^\infty a_ncosn \frac{n\pi x}{l}, 展开式只有余弦函数, 故为余弦级数  \\
在[-l, l]任何一个函数f(x)可以写成一个奇函数和偶函数之和, f(x) = \frac{f(x) - f(-x)}{2} + \frac{f(x) + f(-x)}{2}  \\
则f(x)可以看成一个正弦级数和余弦级数的叠加  \\
(7)延拓: 扩展定义域, 用于f(x)只在[0, l]给出, 需要展开为2l为周期的傅里叶级数  \\
奇延拓: 令F(x) = \begin{cases}
f(x), 0 < x \leqslant l  \\
{-}f(-x), -l \leqslant x < 0  \\
\end{cases}, F(x)是延拓后的奇函数  \\
若原本f(0)不为0, 则这里强行修改令F(0) = 0, 则F(x)为[-l, l]的奇函数  \\
偶延拓: 令F(x) = \begin{cases}
f(x), 0 \leqslant x \leqslant l  \\
f(-x), -l \leqslant x < 0  \\
\end{cases}, F(x)是延拓后的偶函数  \\
延拓后, 奇偶函数就可展开为正弦,余弦级数了  \\
```

# 自我总结
1. 数列是有穷的n项, 级数是无穷的∞项, n项之后还有
2. 收 + 收 = 收, 收 + 发 = 发, 发 + 发 = 不确定
3. 收 * 发 = 不确定, 发 * 发 = 不确定, 但正项级数有收 * 收 = 收, 或者一个正项级数收敛, 其通项的平方的级数也收敛
4. 级数判别可能用到均值不等式
5. 级数的敛散性可以用`等价`是因为级数可以看做有限项和 + 无限项和, 只要无限项和收敛, 级数收敛, 在无限项和时等价的式子与原式子敛散性相同
6. 分母变小可以放缩成大的式子(或者极限有界性, 重要不等式),然后证明收敛(大收敛, 小收敛)
